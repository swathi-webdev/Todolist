#  To-Do List App

This is a simple and responsive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.  
It allows users to add, delete, and manage their daily tasks easily.  
The project focuses on DOM manipulation, event handling, and clean UI design.

##  Features
-  Add new tasks  
-  Delete tasks  
-  Mark tasks as completed  
-  Saves tasks using localStorage
